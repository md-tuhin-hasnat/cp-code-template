Trie (prefix tree) for efficient string storage and prefix-based queries.

\textbf{Time Complexity:} $O(L)$ per operation, where L is word length

\textbf{Operations:}
\begin{itemize}
\item \texttt{insert(word)}: Add word
\item \texttt{countWordsEqualTo(word)}: Count exact matches
\item \texttt{countWordsStartingWith(prefix)}: Count with prefix
\item \texttt{erase(word)}: Remove word
\end{itemize}
