Segment Tree with Lazy Propagation for efficient range queries and range updates.

\textbf{Time Complexity:}
\begin{itemize}
\item Build: $O(n)$
\item Range Query: $O(\log n)$
\item Range Update: $O(\log n)$
\end{itemize}

\textbf{Key Feature:} Lazy propagation delays updates until needed, allowing O(log n) range updates.
