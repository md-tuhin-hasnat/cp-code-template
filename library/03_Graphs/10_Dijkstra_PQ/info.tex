Dijkstra's algorithm finds the shortest path from a source vertex to all other vertices in a weighted graph with non-negative edge weights. This implementation uses a priority queue for efficiency.

\textbf{Time Complexity:}
\begin{itemize}
\item Build: $O(V + E)$
\item Query: $O((V + E) \log V)$
\end{itemize}

\textbf{Space Complexity:} $O(V + E)$

\textbf{Usage:}
\begin{itemize}
\item Call \texttt{init(n)} to initialize for n vertices
\item Add edges using \texttt{graph[u].push\_back(v)} and \texttt{weight[u].push\_back(w)}
\item Call \texttt{dijkstra(source, destination)} to find shortest distance
\item Call \texttt{getPath(source, destination)} to retrieve the path
\end{itemize}
