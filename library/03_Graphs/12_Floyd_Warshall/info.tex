Floyd-Warshall algorithm finds shortest paths between all pairs of vertices in a weighted graph.

\textbf{Time Complexity:} $O(V^3)$

\textbf{Space Complexity:} $O(V^2)$

\textbf{Usage:}
\begin{itemize}
\item Call \texttt{init(N)} to initialize distance matrix
\item Set \texttt{dis[u][v] = w} for each edge
\item Call \texttt{floydWarshall(N)} to compute all-pairs shortest paths
\item \texttt{dis[u][v]} contains shortest distance from u to v
\end{itemize}

\textbf{Applications:}
\begin{itemize}
\item All-pairs shortest path
\item Transitive closure
\item Detecting negative cycles (check if \texttt{dis[i][i] \textless 0})
\end{itemize}
