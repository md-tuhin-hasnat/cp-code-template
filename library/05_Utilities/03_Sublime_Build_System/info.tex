\textbf{Sublime Text Build System for Competitive Programming}

Simple and fast C++ build system with automatic I/O redirection.

\textbf{Setup:}
\begin{itemize}
\item Save as \texttt{CP.sublime-build} in Sublime's User packages folder
\item Location: \texttt{Preferences → Browse Packages... → User/}
\item Select via \texttt{Tools → Build System → CP}
\end{itemize}

\textbf{Usage:}
\begin{itemize}
\item Press \texttt{Ctrl+B} to compile and run
\item Input: \texttt{inputf.in}
\item Output: \texttt{outputf.out}
\item Both files must be in the same directory as your code
\end{itemize}

\textbf{Features:}
\begin{itemize}
\item C++17 standard compilation
\item Automatic I/O redirection
\item Simple and fast
\item Works on Linux (adapt for Windows by using \texttt{.exe})
\end{itemize}
