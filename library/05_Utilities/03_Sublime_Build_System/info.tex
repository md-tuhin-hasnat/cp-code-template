\textbf{Sublime Text Build System for Competitive Programming}

A custom build system for C++ competitive programming with input/output redirection.

\textbf{Setup:}
\begin{itemize}
\item Save this as \texttt{CP.sublime-build} in Sublime's User packages folder
\item Location: \texttt{Preferences → Browse Packages... → User/}
\item Select via \texttt{Tools → Build System → CP}
\end{itemize}

\textbf{Usage:}
\begin{itemize}
\item \texttt{Ctrl+B} - Compile and run with \texttt{input.txt}
\item \texttt{Ctrl+Shift+B} - Select variant (compile only, run only, etc.)
\item Output redirected to \texttt{output.txt}
\item Compilation errors shown in console
\end{itemize}

\textbf{Features:}
\begin{itemize}
\item Fast compilation with \texttt{-O2} optimization
\item Input from \texttt{input.txt}, output to \texttt{output.txt}
\item C++17 standard, includes debug flags
\item Cross-platform (Windows/Linux/macOS variants)
\end{itemize}
