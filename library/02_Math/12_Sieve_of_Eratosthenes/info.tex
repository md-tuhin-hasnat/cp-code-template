Sieve of Eratosthenes generates all prime numbers up to N efficiently.

\textbf{Time Complexity:} $O(N \log \log N)$

\textbf{Space Complexity:} $O(N)$

\textbf{Algorithm:}
\begin{itemize}
\item Mark all multiples of each prime as composite
\item Uses bitset for memory efficiency
\item Generates primes vector for quick access
\end{itemize}
