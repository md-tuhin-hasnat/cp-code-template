Polynomial rolling hash for efficient string matching and substring queries.

\textbf{Time Complexity:}
\begin{itemize}
\item Preprocessing: $O(N)$
\item Substring hash: $O(1)$
\end{itemize}

\textbf{Applications:}
\begin{itemize}
\item Pattern matching (Rabin-Karp)
\item Palindrome checking
\item Counting unique substrings
\end{itemize}
