Knuth-Morris-Pratt (KMP) algorithm for efficient pattern matching in strings.

\textbf{Time Complexity:} $O(N + M)$ where N is text length, M is pattern length

\textbf{Space Complexity:} $O(M)$

\textbf{Key Concept:}
\begin{itemize}
\item Preprocesses pattern to create LPS (Longest Prefix Suffix) array
\item LPS array helps avoid redundant comparisons
\item Never backtracks in the text
\end{itemize}

\textbf{Applications:}
\begin{itemize}
\item String matching
\item Pattern search in text
\item Finding all occurrences of a pattern
\end{itemize}
